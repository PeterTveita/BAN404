# BAN404 – Statistical Learning  
NHH

Course repository for BAN404 at the Norwegian School of Economics.

## Setup

Create environment:

conda env create -f ban404.yml

Activate:

conda activate BAN404

Python version: 3.11

## Structure

- `ban404.yml` – Conda environment  
- `notebooks/` – Jupyter notebooks  
- `src/` – Python scripts  
- `data/` – Local datasets (not version controlled)
